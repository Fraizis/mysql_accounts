## Отчёты по банковским операциям на языке SQL  

### Описание:
- Проект выводит отчёты по банковским операциям на основе таблиц 
приближенных к реальным и тестовых данных
- В проекте использована база данных MySQL
- Для реализации задач использовались запросы на языке SQL

### Структура:
    exc - папка с запросами на языке SQL
    main.py - модуль с подключением к базе данных, созданием таблиц и 
    заполнением их тестовыми данными
    settings.py - данные для подключения к серверу MySQL

### Запуск:
- Находясь в корневой папке проекта установить зависимости командой:

      pip install -r requirements.txt

- Внести данные для подключения к серверу в файле settings.py 
- Запустить файл main.py для создания базы данных и таблиц
- Запустить нужный отчёт в папке exc
