## Отчёты по банковским операциям на языке SQL  

### Описание:
- Проект выводит отчёты по банковским операциям на основе таблиц 
приближенных к реальным и тестовых данных
- В проекте использована база данных SQLite
- Для реализации задач использовались запросы на языке SQL

### Структура:
    exc - папка с запросами на языке SQL
    main.py - модуль с созданием базы данных, 
    таблиц и заполнением их тестовыми данными

### Запуск:
- Запустить файл main.py для создания базы данных и таблиц 
- Запустить нужный отчёт в папке exc
