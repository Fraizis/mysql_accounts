## Отчёты по банковским операциям для проекта "ШИФТ"

### Описание:
- Проект выводит отчёты по банковским операциям на основе таблиц 
приближенных к реальным и тестовых данных
- В проекте использована база данных MySQL 
- Для реализации задач использовались запросы на языке SQL

### Структура:
    exc - папка с запросами на языке SQL
    main.py - модуль с созданием базы данных, 
    таблиц и заполнением их тестовыми данными
    settings.py - модуль с созданием движка для подключения к серверу

### Запуск:
- Находясь в корневом каталоге выполнить команду в терминале:

      pip install -r requirements.txt

- Переименовать файл env.template в .env
- Ввести свои имя пользователя, пароль и сервер для подключения в .env файл 
- Запустить файл main.py для создания базы данных и таблиц 
- Запустить нужный отчёт в папке exc
